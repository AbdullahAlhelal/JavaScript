🔹 Very clear and beginner-friendly

🧬 querySelector()
🧠 Easy Explanation
🔹 Uses CSS selectors
🔹 Returns ONLY the first matching element
🔹 Very flexible

✅ Syntax
document.querySelector("selector");

✅ Examples
document.querySelector("#title");   // by id
document.querySelector(".text");    // by class
document.querySelector("p");        // by tag

⚠️ Important Rule
If many elements match, querySelector() returns:

👉 ONLY the first one

🧬 querySelectorAll()
🧠 Easy Explanation
🔹 Selects ALL matching elements
🔹 Uses CSS selectors
🔹 Returns a NodeList (collection)

✅ Syntax
document.querySelectorAll("selector");

✅ Example
const items = document.querySelectorAll(".text");

items.forEach(el => {
  el.style.color = "orange";
});

📌 Now all elements with class .text are affected.
