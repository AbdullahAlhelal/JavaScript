📖 JSON.parse()
🗝️ Introduction
When data comes from outside your JavaScript code, it usually arrives as JSON text, not as real JavaScript objects.

Examples:

✔ API responses
✔ localStorage data
✔ IndexedDB exports
✔ Configuration files

👉 To use this data in JavaScript, you must convert JSON strings into real JS values.

That’s the role of JSON.parse().


     
🔹 What is JSON.parse()?
JSON.parse() converts a JSON-formatted string into a JavaScript value.

✔ JSON string → JS object / array / value
❌ Invalid JSON → throws an error
